# Components Overview

This directory contains individual documentation for all reusable React components used in the PlayStorm website.

## Core Components

- [**BentoGrid**](./components/BentoGrid.md): Flexible, modern grid layout system.
- [**ConfettiButton**](./components/ConfettiButton.md): Interactive button with a confetti burst effect.
- [**FloatingParticles**](./components/FloatingParticles.md): Atmospheric background particle effect.
- [**Layout**](./components/Layout.md): Global wrapper providing navigation and footer.
- [**LoadingScreen**](./components/LoadingScreen.md): Immersive entrance/loading screen.
- [**MusicPlayer**](./components/MusicPlayer.md): Discreet background audio controls.
- [**ScrollToTop**](./components/ScrollToTop.md): Utility for resetting scroll position on navigation.

## Visual Effects Utility

- [**VisualEffects**](./components/VisualEffects.md): A collection of interactive effects:
    - `CursorTrackerBackground`
    - `ParticleBackground`
    - `GlitchText`
    - `TiltCard`
